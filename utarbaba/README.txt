Read the steps before running UTARBABA website

Steps to run 

1. make sure you have downloaded wampserver 
2. run wampserver
3. insert database.sql to phpMyAdmin to create database
4. make sure the items table has a total of 25 items
5. open localhost/utarbaba in the server and you are
   ready to explore the UtarBaba website
6. Sign up first!
