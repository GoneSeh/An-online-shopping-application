Read the steps before running UTARBABA website

Steps to run 

1. make sure you have downloaded wampserver 
2. download UECS2094_Assign_P2_G07.zip
3. navigate or copy the files inside UECS2094_Assign_P2_G07.zip
   appropriate project directory which is C:\wamp64\www
4. run wampserver
5. insert database.sql to phpMyAdmin to create database
6. make sure hangry_restaurant database has 5 tables
   --> cart, items, guests, orders, order_item
7. make sure the items table has a total of 25 items
8. open localhost/utarbaba in the server and you are
   ready to explore the UtarBaba website
9. Sign up first!

Regards!
